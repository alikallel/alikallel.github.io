HackTheBox: Easy Machine

Posted on May 2024

Enumeration

We start by scanning the target with nmap:

nmap -sC -sV -oA nmap/initial 10.10.10.10

Exploitation

After finding a vulnerable service, we exploit it using the following script:

python3 exploit.py --target 10.10.10.10

Privilege Escalation

We use linpeas.sh to enumerate privilege escalation vectors:

./linpeas.sh

Flag

Finally, we capture the user and root flags:

cat /home/user/user.txt
cat /root/root.txt
