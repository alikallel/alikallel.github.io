Smart Skills Internship: Penetration Testing Experience

Hybrid Internship | June 2025 - September 2025
Penetration Testing Web Security Vulnerability Assessment

About Smart Skills

Smart Skills is an ANCS-certified audit desk specializing in cybersecurity solutions. The company provides comprehensive technical and organizational audit services, along with professional training programs, to both public and private sector clients across Tunisia and internationally. Their mission is to deliver reliable, stable, and secure solutions to enterprises and administrations.

Role Overview

During my hybrid internship as a Penetration Tester, I gained hands-on experience in:

  • Source code security analysis using advanced tools like Snyk
  • Comprehensive vulnerability assessments
  • Detailed security reporting with CVSS scoring
  • Proof of Concept (PoC) development
  • Risk assessment and remediation recommendations
Critical Vulnerabilities
Remote Code Execution (CVSS: 9.9)

Discovered multiple instances of RCE vulnerabilities across different applications, including file upload bypasses and Apache Tomcat exploitation vectors.

SQL Injection (CVSS: 9.1)

Identified SQL injection flaws allowing complete database extraction and manipulation.

High-Severity Issues
Cross-Site Scripting (XSS) - Multiple Variants
  • Stored XSS with CSRF exploitation chains
  • File upload XSS via SVG and HTML files
  • Session hijacking due to missing HttpOnly flags
Authentication Bypass
  • Two-factor authentication bypass via exposed API keys
  • Weak password policies enabling brute force attacks
  • MD5 hash storage vulnerabilities
Access Control Failures
  • Insecure Direct Object References (IDOR)
  • Privilege escalation vulnerabilities
  • Unauthorized access to admin-only resources
Medium-Risk Vulnerabilities
  • Information Disclosure: Exposed debug files, system information, and sensitive configuration data
  • XML-RPC Exposure: Publicly accessible XML-RPC interfaces enabling brute force attacks
  • User Enumeration: Via REST APIs and password reset functionality
  • Spring Boot Actuator Exposure: Heap dumps and HTTP trace endpoints accessible
Low-Risk Issues
  • Outdated WordPress plugins and components
  • Debug mode enabled in production environments
  • Publicly accessible login pages
  • Inadequate error handling revealing system details

Conclusion

My internship at Smart Skills provided invaluable hands-on experience in penetration testing and vulnerability assessment. Working with real-world applications and discovering critical security flaws reinforced the importance of comprehensive security testing in protecting organizations from cyber threats.

The hybrid work model allowed me to experience both collaborative team environments and independent security research. This internship has significantly strengthened my technical skills and prepared me for a career in cybersecurity.
